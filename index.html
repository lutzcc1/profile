Plan-et

This application is for travelers to be able to create their trips and share them with the community. New travelers can take a look at different travel itineraries created by other users to get some inspiration while creating their own.

Developed in Ruby on Rails, it uses Postgres as DBMS, and it's deployed in Heroku. It relies on jQuery and Ajax to provide the best UX while using it.

I did some scrapping to create a small DB that contains restaurants, attractions and hotels from different cities. The app access this info via a self-made API.

Hom-et

This is a web application to share experiences and great moments between locals and tourists. The excuse? A delicious homemade meal.

Foreign tourists can book meals at locals households so that they can enjoy homemade local food while meeting new and exciting people.

On the other side, locals can receive foreign tourists so that they can share their traditions and local expertise while making new friends.

Developed in Ruby on Rails, it uses Postgres as DBMS and it's deployed in Heroku.
